Основы теории чисел

Этот сайт был создан для того, чтобы дать выпускникам, желающим работать в сфере защиты информации, представление об основных фактах теории чисел, которая является базой для создания и использования методов шифрования.

Пусть n>1. Если a-b делится на n, то это означает, что a и b сравнимы по модулю n, то есть имеют одинаковые остатки при делении на n.

Примем n за натуральное число. Тогда функция Эйлера φ(n) – это количество натуральных чисел, взаимно простых с n и не больших n.

Сформулируем теорему Эйлера Если числа a и n взаимно простые, то a в степени φ(n) сравнимо с единицей по модулю n, где φ(n) — функция Эйлера.

Следствием теоремы Эйлера является малая теорема Ферма: Если p –простое число, и a взаимно просто с p, то а в степени p-1 сравнимо с единицей по модулю p.
